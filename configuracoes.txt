1 - Digitar ctrl + shift + p e digitar git init ou clicar na aba e clicar inicializar repositório

2 - Depois abrir o terminal e digitar o comando 

git remote add origin https://github.com/pelocam/loaine-agunlar.git

3 - Instalar o node js

4 - Depois instalar o angular

npm install -g @angular/cli

5- Criar o primeiro projeto

ng new <nome-projeto>

5 - Iniciando o projeto

ng serve

6- No app-module.ts no decorator declarations colocamos os componentes, diretivas e pipes

No providers colocamos os serviços globais para toda a aplicação 

bootstrap colocamos o component princial 

7  - Os módulos servem para organizar o código 

no caso de eu criar um novo component é necessário colocar no módulo na declaração exports quais são os componentes, pipes ou diretivas que desejo exportar 

@NgModule({
  declarations: [
    CursosComponent
  ],
  imports: [
    CommonModule
  ],

  exports:[
    CursosComponent
  ]
})

e depois acrescentar no imports do app.module.ts

8 - ctrl + c para a execução do servidor
